## 🆘 Support & Troubleshooting

### Common Questions

**Q: How do I get the best DPI?**
A: Use mermaid.live, download PNG at highest available resolution (usually 300+ DPI)

**Q: Which tool is easiest?**
A: iLovePDF.com - drag and drop, no installation needed

**Q: Can I modify the diagrams?**
A: Yes! Edit Mermaid code in mermaid.live and re-export

**Q: Do I need to remove all text?**
A: No, keep introductory/concluding sentences. See TEXT_REPLACEMENT_GUIDE.md

**Q: How large will the final PDF be?**
A: Usually 5-15 MB depending on image compression

**Q: Can I use these diagrams elsewhere?**
A: Yes! Export as SVG for use in presentations, websites, etc.

### Troubleshooting

| Issue | Solution |
|-------|----------|
| Diagrams too small | Resize before embedding (see PDF_MODIFICATION_GUIDE.md) |
| Blurry images | Ensure 300+ DPI during export |
| File too large | Compress images using quality=90 setting |
| Wrong position | Manually adjust in PDF editor |
| Missing caption | Add using TEXT_REPLACEMENT_GUIDE.md examples |
